This is the Power Outage Education App for the ECEN403/ECEN404 Senior Capstone project. The developers for this project are Joey Raphael, Aidan Petropoulos, and Jackie Villanueva. This project is sponsored by Dr. Mladen Kezunovic, targeted at teaching students from K-12th about Power Outages.

The main app, intended to be used on Android phones, may be installed from the flutter_application_1 folder, which navigates to all sections of the app.

The app is comprised of 3 subsections: 
-DoSeum Game for Elementary Students (K-5th grade), developed by Aidan Petropoulos in Unity.
-Power Tycoon Game for Middle School Students (6th-8th grade), developed by Joey Raphael in Unity.
-High School Lesson for High School Lessons (9th - 12th grade), developed by Jackie Villanueva using Flutter in Android Studio.
